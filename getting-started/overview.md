# Claude Code Overview

Claude Code is an agentic coding tool that reads your codebase, edits files, runs commands, and integrates with your development tools. Available in your terminal, IDE, desktop app, and browser.

## Get Started

| Environment | Install Command |
|---|---|
| macOS/Linux/WSL | `curl -fsSL https://claude.ai/install.sh \| bash` |
| Windows PowerShell | `irm https://claude.ai/install.ps1 \| iex` |
| Homebrew | `brew install --cask claude-code` |
| WinGet | `winget install Anthropic.ClaudeCode` |

Then: `cd your-project && claude`

## What You Can Do

- Automate tedious tasks (tests, lint, merge conflicts, release notes)
- Build features and fix bugs with natural language
- Create commits and pull requests via git integration
- Connect external tools with MCP (Model Context Protocol)
- Customize via CLAUDE.md, skills, and hooks
- Run agent teams and custom agents (Agent SDK)
- Pipe, script, and automate from the CLI (Unix philosophy)
- Schedule recurring tasks (cloud or desktop)
- Work from anywhere (Remote Control, iOS app, /teleport)

## Use Claude Code Everywhere

| I want to… | Best option |
|---|---|
| Continue a local session from my phone | Remote Control |
| Push events from Telegram/Discord/webhooks | Channels |
| Start locally, continue on mobile | Web or Claude iOS app |
| Run on a recurring schedule | Cloud or Desktop scheduled tasks |
| Automate PR reviews | GitHub Actions or GitLab CI/CD |
| Auto code review on every PR | GitHub Code Review |
| Route bug reports from Slack to PRs | Slack integration |
| Debug live web applications | Chrome extension |
| Build custom agents | Agent SDK |
