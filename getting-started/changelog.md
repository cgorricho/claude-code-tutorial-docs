## 2.1.91 — April 2, 2026
- MCP tool result persistence override via `_meta["anthropic/maxResultSizeChars"]` (up to 500K) for large results like DB schemas
- `disableSkillShellExecution` setting to disable inline shell execution in skills/slash commands/plugins
- Multi-line prompts in `claude-cli://open?q=` deep links now supported
- Plugins can ship executables under `bin/` and invoke them from the Bash tool
- Fixed transcript chain breaks on `--resume`
- Fixed `cmd+delete` on iTerm2, kitty, WezTerm, Ghostty, Windows Terminal
- Fixed plan mode in remote sessions after container restart
- Edit tool uses shorter `old_string` anchors, reducing output tokens

## 2.1.90 — April 1, 2026
- `/powerup` — interactive lessons teaching Claude Code features with animated demos
- `CLAUDE_CODE_PLUGIN_KEEP_MARKETPLACE_ON_FAILURE` env var for offline environments
- Fixed infinite loop in rate-limit dialog
- Fixed `--resume` prompt-cache miss with deferred tools
- Fixed auto mode ignoring user boundaries ("don't push", "wait for X before Y")
- Hardened PowerShell permission checks
- `/buddy` — April 1st easter egg, hatch a creature that watches you code

## 2.1.89 — April 1, 2026
- `"defer"` permission decision in `PreToolUse` hooks — headless sessions can pause and resume with `-p --resume`
- `CLAUDE_CODE_NO_FLICKER=1` for flicker-free rendering
- `PermissionDenied` hook — return `{retry: true}` to allow retry after auto mode denial
- Named subagents now appear in `@` mention typeahead
- `MCP_CONNECTION_NONBLOCKING=true` for `-p` mode to skip MCP wait
- Fixed CRLF doubling on Windows in Edit/Write tools
- Fixed `StructuredOutput` schema cache (~50% failure rate with multiple schemas)
- Thinking summaries no longer generated by default (set `showThinkingSummaries: true` to restore)
- `/env` now applies to PowerShell tool commands
